Go to dashboard

The dashboard runs 100% locally — you open it from your terminal.

Note When The Framework is running, refresh this page — you'll be redirected to the Dashboard (the-framework.local).

Run

If The Framework is installed, run it:

$ the-frameworkcopy
Install

Not installed yet? Install it globally:

$ npm i -g @gemstack/the-frameworkpnpm add -g @gemstack/the-frameworkbun add -g @gemstack/the-frameworkyarn global add @gemstack/the-frameworkcopy
One-time run

You just want to try it out? Run it once, no install:

$ npx @gemstack/the-frameworkpnpm dlx @gemstack/the-frameworkbunx @gemstack/the-frameworkyarn dlx @gemstack/the-frameworkcopy

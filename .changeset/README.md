# Changesets

This directory contains changesets for GemStack releases.

## How to create a release

```bash
pnpm changeset            # describe the change + pick the bump
pnpm changeset:version    # apply versions + update CHANGELOGs
pnpm release              # build + publish to npm
```

See the [changesets docs](https://github.com/changesets/changesets) for details.
